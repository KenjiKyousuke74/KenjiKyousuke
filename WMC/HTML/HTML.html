Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat struktur dan konten dari halaman web, HTML menggunakan serangkaian elemen dan tag untuk mendefinisikan bagian-bagian yang berbeda dalam halaman web.

Elemen HTML direpresentasikan oleh tag yang dikelilingi oleh tanda kurung sudut, Tag HTML juga dapat memiliki atribut yang memberikan informasi tambahan tentang elemen tersebut, Atribut ditentukan dalam tag pembuka dan dapat memodifikasi perilaku atau tampilan elemen tersebut.

Dokumen HTML memiliki struktur dasar sebagai berikut:

html
Tag HTML Fungsi Keterangan
<h1> - <h6> Heading Menandai judul atau heading dengan level yang berbeda
<p> Paragraf Membuat paragraf teks
<a> Tautan Membuat tautan ke halaman atau sumber lain
<img> Gambar Menampilkan gambar di halaman web
<ul> Daftar tak-berurutan Membuat daftar tak-berurutan
<ol> Daftar berurutan Membuat daftar berurutan
<table> Tabel Membuat tabel untuk menampilkan data terstruktur
<form> Formulir Membuat formulir interaktif untuk mengumpulkan data
<div> Kontainer Memisahkan dan mengelompokkan elemen-elemen HTML
<span> Elemen Inline Membuat konten dengan gaya tertentu dalam teks
